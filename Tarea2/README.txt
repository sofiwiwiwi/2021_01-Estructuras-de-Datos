Para la compilación del archivo problema_n.cpp se deben poner el siguiente comando en la carpeta Parte-n:

	g++ problema_n.cpp -Wall -o <nombre-ejecutable>

Y para la posterior ejecución se debe usar:
	
	./<nombre-ejecutable> < entradaPolinomio.txt

Reemplazando n con 1 o 2 según corresponda.

El resultado esperado se encontrará en salidaPolinomio.txt en la carpeta Parte-n correspondiente.
